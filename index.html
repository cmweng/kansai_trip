關於本站
在今年(2019) 11月底和朋友一同到關西自助旅行，1天大叛、1天姬路城、4天京都，運氣不錯碰到常寂光寺紅葉見煩，彌補了2015年沒看到紅葉的缺憾。因此寫個網頁做記錄，包含行程、交通、飲食等資訊，以下是這次的行程
  • Day1 TPE → KIX → 難波 → 大阪城 → 阿倍野HARUKAS300展望台 → 首頓堀 → 旅館
  • Day2 難波 → 梅田車站轉JR → 姬路城 → 京都 → 京都車站 → 旅館
  • Day3 京都車站 → JR嵐山站 → 常寂光寺 → 西芳寺 → 金閣寺 → 北野天滿宮 → 京都車站 → 旅館
  • Day4 京都車站 → 伏見稻荷大社 → 宇治 → 平等院 → 宇治上神社 → 宇治神社 → 京都車站 → 旅館
  • Day5 旅館 → 東寺 → 購物行程 → 旅館
  • Day6 中央郵局 → 京都車站 → KIX → TPE

大阪城

由於第一天移動日時間有限,到難波時已中午,吃完午餐第一個點就是大阪城,天氣不是好,陰陰的,遊客不少,這邊紅葉大都掉光,不然就是乾枯,我們並未上天守閤,只在外面拍拍照,就前往下個點(話雖如此,也花了一個多小時)。

阿倍野HARUKAS 300展望台

顧名思義,來展望台就是來看景的,展望台在第58~60樓,第60樓360°都是落地窗,可以環看大阪市,晚上來這看夜景很棒,只過票價不便宜要1500日元,不限時間,如果白天走累來,晚上來這放空看夜景也很棒,另外在從16樓搭電梯上展望台的過程也是一大亮點,電樣內部有聲光投影,也很精彩。

姬路城

與熊本城、松本城被稱為日本三大名城,姬路城又稱白鷺城,相當美的城。由JR車站一出來,就可以看到姬路城,步行約10分鐘就可以抵達,接近姬路城時,人行道兩側種了銀杏,樹葉已轉黃,向前望向姬路城,眼詩情畫意,這次由於時間的關係,沒上天守閤,下次找個櫻花季再來訪吧。

常寂光寺

嵐山賞楓著名景點之一,此行碰巧,真的是碰巧,楓紅時間往常慢了點,因此遇到常寂光寺紅葉見頃,一早到常寂光寺,在入口處即有數顆楓樹轉紅,紅綠黃三色交織,加上晨間的陽光照射,真的美極了。

西芳寺

被人稱京都最難造訪的世界遺產,之所以會被這樣形容,主要是它採預約制拜觀,需要在一個月之前以手寫郵件預約,並附上回郵名信片給寺方回覆您預約結果,另一方面拜觀費用3000日元也算是拜觀費用最高的寺院,另外拜觀時需抄寫經文也是它的特色。西芳寺另一個名稱叫苔寺,顧名思義,在它的庭園裡,地上長滿多種不同種類的青苔,相當特別,這也是遊客造訪的主要原因。

金閤寺

這算是到京都旅遊必訪景點之一,但說實話,點有點遠,建議行程安排要斟酌,建議造訪的時間點是午後,甚至傍晚,最晚購票入寺時間是16：30,午後金閣寺是順光方向,在陽光照射下反射金色光芒,背景也因順光,較不會一片死白。網頁上金閤寺的照片是2015年拍攝,這次到金閤寺,遊客爆量,卡不到好位置拍聶,且景也沒2015年時漂亮,所以就放上2015年的照片。

北野天滿宮

這算是一個意外的景點因爲離金閣寺很近,又剛好遇到紅葉見頃,因此排在金閣寺後,直接前往夜間拜觀,夜晚的紅葉打上燈光,別有一番美感,可惜這次旅行沒帶腳架,手持拍攝的照片大都手震糢糊。

伏見稻荷大社

應該大部份人去京都都是為了千本鳥居而來,伏見稻荷大社遊客人潮相當的多，建議如果行程許可，可以一大早就來，可避開人潮，體力許可的話，爬一下後山，是相當不錯的旅行體驗。

宇治平等院

建於西元1053年,主要建築是阿彌陀堂,因置奉阿彌陀如來與52尊雲中供奉菩薩像而得名,後因建築外型像欲振翅而飛的禽鳥,加上中堂脊沿有兩隻金銅鳳凰像,在江戶時代改名為鳳凰堂,在現今10圓硬幣上有鳳凰堂的圖案,100000圓紙鈔背面有鳳凰圖案,鳳凰堂在1994年被指定為世界文化遺產。

宇治上神社

宇治上神社本殿建於西元1060年,為「現存最古老的神社建築」,亦登錄為世界文化遺產。神社不大,可能來訪的遊客相對其他景點少,顯得特別靜謐,特別喜歡鳥居到拜殿間的參道,兩側的楓正是由綠轉紅的階段,下午的斜射光照射在葉子上顏色相當漂亮,走在這段參道上,相當舒服放鬆。

東寺

東寺建於西元796年,歷經多次損毀重建,於1994年登錄為世界遺產 ,購票入內可參觀金堂、講堂內部,五重塔部分則是在底層四面打開窗戶,讓遊客由窗戶向內參觀。如果和我一樣,主要是去拍五重塔的,建議可叢排下午造訪,下午拍攝五重塔就是順光的方向,拍出來就不會黑黑的。
